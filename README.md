# Flashcards 

**Flashcards web app** that helps users study by flipping through questions and answers. This project was created to practice JavaScript, DOM manipulation, and basic UI interaction.

**Live Demo:** https://kelseysemien.github.io/flashcards/

---

## About

This flashcards app displays a set of cards with questions on the front. Clicking a card flips it to reveal the answer. Users can navigate through the flashcards one by one.

It’s built using **HTML, CSS, and JavaScript** and hosted for free on **GitHub Pages**. This project highlights dynamic UI behavior using DOM events and JavaScript logic.

---

## Features

 Displays a stack of flashcards  
 Click a card to flip between question and answer  
 Navigate between cards with “Next” and “Previous” controls  
 Clean, minimal interface  

---

## Tech Stack

This project uses:

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript** – Logic & DOM  
- **GitHub Pages** – Hosting

---

## How It Works

1. The flashcards are defined in a JavaScript array.
2. The app displays one card at a time.
3. Clicking the card flips it to show the answer.
4. “Next” and “Previous” buttons update which card is shown.
